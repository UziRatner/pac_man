# מחשבון רופי ↔ שקל 💱

אפליקציית ווב בקובץ יחיד להמרת מחירים בין **רופי הודי (INR)** ל**שקל חדש (ILS)**, עם שער חליפין חי.

## שימוש מהיר
פתחו את `index.html` בכל דפדפן (מחשב או טלפון). אין צורך בהתקנה.

## התקנה על מסך הבית של הטלפון (PWA) 📱
האפליקציה בנויה כ-**PWA** — אפשר להתקין אותה כאייקון על מסך הבית, לפתוח במסך מלא ולהשתמש גם בלי אינטרנט.

> ⚠️ **חשוב:** התקנה עובדת רק כשהדף מוגש דרך **HTTPS** (לא מקובץ מקומי). הדרך הפשוטה — GitHub Pages.

**הפעלת אירוח דרך GitHub Pages:**
1. מזגו את הענף ל-`main`.
2. ב-GitHub: `Settings → Pages → Source: Deploy from a branch → main → /(root)`.
3. הכתובת תהיה: `https://uziratner.github.io/pac_man/rupee-shekel-converter/`

**התקנה בטלפון:**
- **אנדרואיד (Chrome):** פותחים את הכתובת → מופיע כפתור "התקן על מסך הבית", או תפריט ⋮ → "התקן אפליקציה".
- **אייפון (Safari):** כפתור שיתוף ⬆️ → "הוסף למסך הבית".

## תכונות
- **המרה דו-כיוונית** — כפתור החלפה (⇅) הופך בין רופי↔שקל.
- **שער חליפין חי** — נשלף אוטומטית מ-API ציבורי חינמי (ללא מפתח).
  - מקור ראשי: [Frankfurter](https://frankfurter.dev)
  - גיבוי: [open.er-api.com](https://open.er-api.com)
- **עבודה לא-מקוונת מלאה** — Service Worker שומר את האפליקציה, והשער האחרון נשמר מקומית ומוצג גם ללא אינטרנט.
- **שער ידני** — אפשר להזין שער חליפין באופן ידני כשאין רשת.
- **חישוב חי תוך כדי הקלדה**, כפתורי סכום מהירים, וזיכרון של הסכום האחרון.
- **עיצוב מותאם** — RTL בעברית, מצב כהה/בהיר אוטומטי, מותאם לנייד.

## פרטיות
כל החישוב מתבצע בדפדפן. הנתונים היחידים היוצאים החוצה הם בקשת שער החליפין ל-API.
