# מחשבון רופי ↔ שקל 💱

אפליקציית ווב בקובץ יחיד להמרת מחירים בין **רופי הודי (INR)** ל**שקל חדש (ILS)**, עם שער חליפין חי.

## שימוש
פתחו את `index.html` בכל דפדפן (מחשב או טלפון). אין צורך בהתקנה או שרת.
כדי לשמור לגישה מהירה: הוסיפו את הדף למסך הבית / סמנו כמועדף — האפליקציה נשארת פתוחה ונוחה לשימוש יומיומי.

## תכונות
- **המרה דו-כיוונית** — כפתור החלפה (⇅) הופך בין רופי↔שקל.
- **שער חליפין חי** — נשלף אוטומטית מ-API ציבורי חינמי (ללא מפתח).
  - מקור ראשי: [Frankfurter](https://frankfurter.dev)
  - גיבוי: [open.er-api.com](https://open.er-api.com)
- **עבודה לא-מקוונת** — השער האחרון נשמר מקומית ומוצג גם ללא אינטרנט.
- **שער ידני** — אפשר להזין שער חליפין באופן ידני כשאין רשת.
- **חישוב חי תוך כדי הקלדה**, כפתורי סכום מהירים, וזיכרון של הסכום האחרון.
- **עיצוב מותאם** — RTL בעברית, מצב כהה/בהיר אוטומטי, מותאם לנייד.

## פרטיות
כל החישוב מתבצע בדפדפן. הנתונים היחידים היוצאים החוצה הם בקשת שער החליפין ל-API.

> הערה: זהו פרויקט נפרד ואינו קשור למשחק ה-Pac-Man שבשורש המאגר.
